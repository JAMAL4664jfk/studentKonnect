TABLES CREATED:
---------------

1. accommodations
   - Student housing listings
   - Supports multiple property types
   - Geolocation support
   - JSON amenities and images

2. marketplaceItems
   - Buy/sell items between students
   - Multiple categories and conditions
   - View counter and featured flag
   - JSON images array

3. rewards
   - User loyalty points tracking
   - 4-tier level system (bronze to platinum)
   - Lifetime earning/redemption tracking

4. rewardTransactions
   - Point earning and redemption history
   - Reference tracking to related entities

5. rewardCatalog
   - Available rewards for redemption
   - Multiple categories
   - Stock management
   - Terms and conditions
